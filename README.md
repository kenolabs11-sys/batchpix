# BatchPix

Offline batch image converter.

## Features
- Drag & drop images or folders
- Batch convert
- Export: PNG / JPG / WEBP
- Resize (width / height / percent)
- Works fully offline

## Why BatchPix?
- No upload
- No account
- No tracking
- Fast and simple
